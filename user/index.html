How to use Meeting App

1. To get Started, on the Main Desk, we can see the 'Tools' icon, click on it.

As we can see, under the Tools list, 'Meeting' option is available. Click on it.

2. Meeting view

This is the default view that'll display all the Meetings that've been set.

-> Calendar View We can also view the Meetings in Calendar view that can be selected from left-hand side toolbar.

3. Set up a new Meeting

1. Plan a Meeting

2. Prepare an Agenda

3. Add Attendees

Note: Attendees must be Users in the System

-> How to add User in the System

In the Search bar type 'User List' and click on the option that is displayed in dropdown

The List will display existing Users. On the top right-hand side, 'New' button will be seen. Click on it and add a new-user to the System.As soon as a new User is added, an Orientation Meeting for that user is automatically added in the Meeting' List.
